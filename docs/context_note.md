## 1. 서비스 개요 및 비즈니스 모델

### 브랜드 정체성
- **서비스명**: 핀테이블 (FinTable)
- **핵심 가치**: "복잡한 금융 정보를 테이블 위에 깔끔하게 정리하여 보여준다"
- **슬로건**: "지금 가장 좋은 금리, 한눈에 비교하세요"
- **디자인 톤**: 따뜻하고 깊이 있는 베이지(`#F5F3EF`) 배경, **Pretendard** 폰트 기반의 볼드하고 프리미엄한 타이포그래피, 고해상도 글래스모피즘 카드 UI.

### 서비스 구성
1. **메인 (금융상품)**: 금감원 API 기반 예적금 비교.
2. **Treia (서브)**: 골드 EA 자동매매 정보 및 트랙레코드.
3. **Housing (부동산)**: LH/SH 임대주택 공고 큐레이션 (3차 목표).
4. **Loan Diagnostic (신규)**: 뱅크샐러드 스타일의 가상 대출 진단 프로세스.

### 3. 주요 업데이트 이력

#### [2026-03-22] 긴급 UX 및 데이터 개선 (Emergency Refinement)
- **가독성 및 대비 강화**: 텍스트 컬러를 더 어두운 톤으로 조정하고 배경색(`#F5F3EF`)을 보정하여 시각적 피로도 감소 및 콘텐츠 집중도 향상.
- **데스크탑 최적화**: 메인 콘텐츠 너비를 `max-w-5xl`로 제한하여 가독성이 좋은 최적의 뷰포트 확보.
- **대출 데이터 무결성**: API에서 넘어오는 0% 금리 데이터를 `null`로 처리하고 정렬 로직에서 제외하여 신뢰도 높은 정보 제공.
- **카드 섹션 고도화**: AI로 생성한 프리미엄 카드 이미지(`Deep Dream`, `M Edition`, `다담`, `iD ON`)를 적용하고 상세 혜택 데이터를 보강함.
- **가상 진단 프로세스 (`LoanDiagnosticFlow`)**: 로그인 없이도 뱅크샐러드의 정교한 대출 탐색 과정을 경험할 수 있는 인터랙티브 모달 시스템 구축.
- **모바일 반응형 수정**: `CompactLoanCard`를 모바일에서 세로 레이아웃으로 전환하고, 시뮬레이터 탭에 가로 스크롤(`whitespace-nowrap`)을 적용하여 사용성 개선.
- **다이렉트 랜딩 강화**: `DIRECT_PRODUCT_URLS` 매핑을 확대하여 구글 검색이나 메인 홈 대신 실제 상품 가입 페이지로 연결되는 링크 비중 확대.
