# 핀테이블(FinTable) 맥락 노트 (Context Note)

---

## 1. 서비스 개요 및 비즈니스 모델

### 브랜드 정체성
- **서비스명**: 핀테이블 (FinTable)
- **핵심 가치**: "복잡한 금융 정보를 테이블 위에 깔끔하게 정리하여 보여준다"
- **슬로건**: "지금 가장 좋은 금리, 한눈에 비교하세요"
- **디자인 톤**: 따뜻하고 깊이 있는 베이지(`#F5F3EF`) 배경, **Pretendard** 폰트 기반의 볼드하고 프리미엄한 타이포그래피, 고해상도 글래스모피즘 카드 UI.

### 서비스 구성
1. **메인 (금융상품)**: 금감원 API 기반 예적금 비교.
2. **Treia (서브)**: 골드 EA 자동매매 정보 및 트랙레코드.
3. **Housing (부동산)**: LH/SH 임대주택 공고 큐레이션 (3차 목표).
4. **Loan Diagnostic (신규)**: 뱅크샐러드 스타일의 가상 대출 진단 프로세스.

## 2. 기술적 사양 및 데이터 전략

### 데이터 아키텍처
| 영역 | 데이터 소스 | 업데이트 주기 |
|------|------------|--------------|
| **금융상품** | 금감원 금융상품통합비교공시 API | 월 1회 공시 (매시간 캐시 갱신) |
| **카드 데이터** | 카드사 다이렉트 데이터 시뮬레이션 | 상시 업데이트 |
| **대출 데이터** | FSS API + 필터링 보정 로직 | 실시간 (0% 금리 제외 로직 포함) |

### 기술 스택
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4 + Vanilla CSS Hybrid
- **Animation**: Framer Motion (고성능 로딩, 탭 전환, **Diagnostic Flow** 애니메이션)
- **Image Generation**: AI 기반 프리미엄 카드 에셋 생성 (`generate_image` 활용)

### 3. 주요 업데이트 이력

#### [2026-03-18] Treia 랜딩페이지 개편 및 무비용 리드 자동화 (Treia Funnel Overhaul)
- **독립 레이아웃 (Layout Isolation)**: `LayoutWrapper`를 통해 FinTable과 Treia의 디자인 컨텍스트를 분리하고 다크 테마 적용.
- **마케팅 퍼널 고도화**: 네거티브한 워딩 제거, 프리미엄 금융 SaaS 용어('Treia 자동매매 프로그램', '모니터링 계좌')로 통일 후 5단계 설득 논리(왜 Gold CFD인가, 파멸 방어 로직 등) 강화.
- **무비용 백엔드 파이프라인 (No-Cost Backend Automation)**: Zapier/Make 없이 Next.js Serverless API(`/api/lead`) + 파이어베이스(`treia_leads` DB) + 텔레그램 Bot API 통신을 활용하여 이메일 중복 체크 및 365일 실시간 관리자 알림(Lead Gen) 완비.

#### [2026-03-22] 긴급 UX 및 데이터 개선 (Emergency Refinement)
- **가독성 및 대비 강화**: 텍스트 컬러를 더 어두운 톤으로 조정하고 배경색(`#F5F3EF`)을 보정하여 시각적 피로도 감소 및 콘텐츠 집중도 향상.
- **데스크탑 최적화**: 메인 콘텐츠 너비를 `max-w-5xl`로 제한하여 가독성이 좋은 최적의 뷰포트 확보.
- **대출 데이터 무결성**: API에서 넘어오는 0% 금리 데이터를 `null`로 처리하고 정렬 로직에서 제외하여 신뢰도 높은 정보 제공.
- **카드 섹션 고도화**: AI로 생성한 프리미엄 카드 이미지(`Deep Dream`, `M Edition`, `다담`, `iD ON`)를 적용하고 상세 혜택 데이터를 보강함.
- **가상 진단 프로세스 (`LoanDiagnosticFlow`)**: 로그인 없이도 뱅크샐러드의 정교한 대출 탐색 과정을 경험할 수 있는 인터랙티브 모달 시스템 구축.
- **모바일 반응형 수정**: `CompactLoanCard`를 모바일에서 세로 레이아웃으로 전환하고, 시뮬레이터 탭에 가로 스크롤(`whitespace-nowrap`)을 적용하여 사용성 개선.
- **다이렉트 랜딩 강화**: `DIRECT_PRODUCT_URLS` 매핑을 확대하여 구글 검색이나 메인 홈 대신 실제 상품 가입 페이지로 연결되는 링크 비중 확대.

#### [2026-03-24] EA 라이선스 관리 시스템 V2 및 실시간 관제 API 구축 (Major Milestone)
- **차세대 어드민 대시보드 (`/admin`)**:
  - **이원화 탭 시스템**: 기존 신청자 관리(Leads)와 신규 라이선스 관리(Licenses)를 분리하여 운영 효율성 극대화.
  - **라이선스 동적 제어**: MT5 계좌번호별 활성화, 만료일 설정, 최대 랏(Max Lot) 한도, 티어(Observer/Pro) 관리 기능 완비.
  - **수익 공유 기반 마련**: 정산 증명을 위한 **투자자 비밀번호(Investor Password)** 저장 필드 및 누적 수익 모니터링 필드 추가.
- **실시간 EA 연동 API 구축**:
  - **라이선스 검증 (`/api/license/check`)**: EA 구동 시 서버에 접속하여 권한 및 랏 제한 수치를 0.1초 만에 확인하는 보안 통로 개설.
  - **상태 보고 (`/api/license/update`)**: EA가 실시간 잔고(Balance), 증거금(Equity), 누적 수익(Profit)을 서버로 전송하여 어드민에서 실시간 확인 가능.
  - **멀티 타임프레임(MTF) 전략 대응**: 한 계좌 내 M2~M15 등 여러 전략이 동시에 구동되더라도 **매직넘버별 성적을 개별 집계**하는 Map 구조 데이터 설계 적용.
- **랜딩 페이지 UX/UI 정밀 소탕**:
  - **애니메이션 서사 최적화**: 인포그래픽 노출 시 차트 곡선 -> 지표 카드 순으로 0.6초 간격의 리드미컬한 순차 노출(Staggered Animation) 튜닝 완료.
  - **레이아웃 결함 수정**: 상단 복귀(ScrollToTop) 시 로고 잘림 현상 해결(여백 조정), 스크롤 유도선 위치 보정, 탭 타이틀 명칭(Treia Gold) 통일.
