## 1. 서비스 개요 및 비즈니스 모델

### 브랜드 정체성
- **서비스명**: 핀테이블 (FinTable)
- **핵심 가치**: "복잡한 금융 정보를 테이블 위에 깔끔하게 정리하여 보여준다"
- **슬로건**: "지금 가장 좋은 금리, 한눈에 비교하세요"
- **디자인 톤**: 따뜻하고 깊이 있는 베이지(`#F5F3EF`) 배경, **Pretendard** 폰트 기반의 볼드하고 프리미엄한 타이포그래피, 고해상도 글래스모피즘 카드 UI.

### 서비스 구성
1. **메인 (금융상품)**: 금감원 API 기반 예적금 비교.
2. **Treia (서브)**: 골드 EA 자동매매 정보 및 트랙레코드.
3. **Housing (부동산)**: LH/SH 임대주택 공고 큐레이션 (3차 목표).
4. **Loan Diagnostic (신규)**: 뱅크샐러드 스타일의 가상 대출 진단 프로세스.

## 2. 기술적 사양 및 데이터 전략

### 데이터 아키텍처
| 영역 | 데이터 소스 | 업데이트 주기 |
|------|------------|--------------|
| **금융상품** | 금감원 금융상품통합비교공시 API | 월 1회 공시 (매시간 캐시 갱신) |
| **카드 데이터** | 카드사 다이렉트 데이터 시뮬레이션 | 상시 업데이트 |
| **대출 데이터** | FSS API + 필터링 보정 로직 | 실시간 (0% 금리 제외 로직 포함) |

### 기술 스택
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4 + Vanilla CSS Hybrid
- **Animation**: Framer Motion (고성능 로딩, 탭 전환, **Diagnostic Flow** 애니메이션)
- **Image Generation**: AI 기반 프리미엄 카드 에셋 생성 (`generate_image` 활용)
