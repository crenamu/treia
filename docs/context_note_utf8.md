# 트레이아(Treia) 맥락노트 (Context Note)

## 1. 프로젝트 배경 및 목적

- **작성일**: 2026.02.28 15:30 (최종 업데이트: 2026.03.09 10:45)
- **배경**: 12년 경력의 트레이더로서, 입문자들이 사기 혹은 불필요한 시행착오로 돈과 시간의 손실을 겪는 것을 막기 위해 신뢰할 수 있는 정보를 제공함.
- **핵심 목표**: 정보의 큐레이션 및 AI를 통한 번역・요약 기능을 통해 입문자가 스스로 매매 기준을 세울 수 있도록 돕는 플랫폼.
- **주요 컨셉 (추가 확정 예정)**:
  1. **자동매매(EA) 가이던스**: 사용자 스스로 EA를 제작/활용할 수 있도록 돕는 팩토리 역할.
  2. **데이터 기반 카피트레이딩**: MQL5 시그널 기반의 투명하고 객관적인 리딩/카피 지원.

## 2. 개발 상세 가이드 (스타일/기능)

### 2.1 Next.js 14 설정

- **기본 구조**: 루트 하위에 `app`, `components`, `lib` 배치, App Router 사용.

### 2.2 UI/UX 디자인 정책 (Premium Look)

- **Primary Theme**: Deep Dark (#0B0C10)
- **Accent Color**: Gold (#D4AF37) / Amber (#FFBF00) -> 주된 강조색
- **개선 목표**: 교육 자료 내 풍부한 시각성(인라인 이미지, 패턴 이미지 등) 제공. 인사이트 카드 리스트 가로/세로 스크롤 적용 처리.

### 2.3 데이터 인프라 및 API

- **Firestore**: 단독 DB `treia` 구축 완료 (`treia_` 접두사로 구분 및 인스턴스 명시).
- **Economic Calendar**: Forex Factory 데이터 기반으로 동작. Gemini AI를 거쳐 영문 명칭을 한글로 번역 후 서비스. 번역 API 문제 발생 시를 대비해 원본 영문을 반환하는 Fallback 예외처리(Try/Catch) 적용 완료 (2026.03.09).
- **Vision AI (OCR) 매매일지 생신기**: 향후 사용자가 캡처한 MT4/MT5 거래 내역 이미지를 업로드 시 Gemini Vision API를 통해 데이터 추출/변환.

## 3. 리서치 및 참고 자료

- MQL5 시그널, Forex Factory OpenAPI.
- MTF(Multi-Timeframe) 데이터베이스 연산 및 차트 매퍼.

## 4. 특이 사항 및 차별점 분석 (EA + Copy Trading Concept)

- **장점**: 초보자(카피트레이딩)와 중/고급자(EA 제작/개선) 트래픽을 동시에 유입시킬 수 있는 리텐션 극대화 파이프라인.
- **EA Builder (노코드 템플릿 도입)**: 코딩을 모르는 사용자도 지표, 캔들 패턴(도지, 장악형 등)을 조합하여 MQL5 EA를 생성할 수 있도록 `app/ea/builder` 페이지 프리미엄 UI 설계 완료 (2026.03.09). 향후 사용자가 선택한 조합(RSI + 도지캔들 등)을 프롬프트로 변환하여 AI가 코드 초안을 짜주는 방향으로 연동.
- **고려 사항**: MQL4/5 등 EA 제작 허들을 낮출 수 있는 'AI 기반 EA 추천 프롬프트'나 '개념 가이던스'가 핵심 킬러앱이 될 수 있음. 시그널(카피)에 대해서는 플랫폼의 객관성(수익성 보장 금지)을 철저히 유지하는 것이 필수 법적 보호 장치.
