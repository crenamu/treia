# 핀테이블(FinTable) 실행 계획서 (Implementation Plan)

_최종 업데이트: 2026-03-13 (금) 14:00 KST_

---

## 1. 현재 상황 분석 (Pivot: Treia → FinTable)

- **배경**: 골드 EA 전문 서비스(Treia)에서 종합 금융 정보 큐레이션 플랫폼(FinTable)으로 확장.
- **핵심 목표**: "공공데이터(금감원, LH/SH) 기반의 투명한 금융 정보 제공 및 Treia의 서브 메뉴 내재화."
- **브랜딩**: 옅은 베이지(`#FBF9F6`) 바탕의 깔끔하고 신뢰감 있는 핀테크 UX.

---

## 2. 릴리즈 단계 및 로드맵

### Phase 1: MVP - 금융상품 비교 플랫폼 (2026년 4월 목표)

**목표**: 금감원 API 연동을 통한 예금/적금 금리 비교 서비스 런칭.

- **UI 구현**: 
    - 옅은 베이지 테마의 메인 대시보드 (`/`)
    - 상품 요약 카드 및 기간별 필터 시스템
    - 상품 상세 정보 페이지 (`/deposit/[id]`)
- **API 연동**: 
    - 금감원 금융상품 통합 비교 공시 API 연동
    - 서버 사이드 캐싱 전략 적용
- **Treia 이식**: 
    - 기존 Treia 랜딩 페이지를 `/treia` 경로로 이동 및 메뉴화

### Phase 2: 서비스 고도화 및 개인화 (2026년 6월 목표)

- **인증 시스템**: Firebase Auth 기반 소셜 로그인 (구글/카카오)
- **개인화 기능**: 관심 상품 저장, 알림 설정
- **컨텐츠 확충**: 
    - 적금 상품 비교 정식 오픈
    - Treia 실적 게시판(Track record) 연동

### Phase 3: 영역 확장 및 수익화 (2026년 9월 이후)

- **부동산**: LH/SH 임대주택 모집공고 비교 (`/housing`)
- **투자**: ETF, ISA 상품 비교 추가
- **수익화**: 금융사 제휴 (CPC), 프리미엄 알림 구독 서비스

## 3. 핵심 아키텍처 및 폴더 구조 계획

```text
app/
├── (fintable)/      ← 메인 금융 서비스 (Light 테마)
│   ├── page.tsx     ← 예금 리스트
│   └── [id]/        ← 상품 상세
├── treia/           ← 골드 자동매매 (Dark 테마 유지)
│   └── page.tsx
├── housing/         ← 임대주택 (3차 예정)
│   └── page.tsx
└── api/
    ├── deposits/    ← 금감원 API 프록시
    └── housing/     ← LH/SH API 연동
```
