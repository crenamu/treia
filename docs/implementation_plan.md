## 1. 현재 상황 분석

- **배경**: 카피트레이딩 + EA 교육 플랫폼으로 방향을 잡고, 홈페이지 랜딩 완성도를 높이는 중.
- **핵심 목표**: "EA 입문자를 설득하고, 트랙레코드로 신뢰를 쌓고, 카피트레이딩으로 수익을 연결하는 퍼널 구현"
- **법적 포지셔닝**: Treia = EA 소프트웨어 교육 SaaS. 투자 권유 없음. 퍼포먼스 피는 브로커 플랫폼을 통해 수취.
