# 트레이아(Treia) 실행 계획서 (Implementation Plan)

## 1. 현재 상황 분석

- **완료**: MT5 로컬 데이터 클라우드(Firebase) 연동, Vercel 초고속 로딩 최적화, UI 높이 정렬.
- **이슈**: 텔레그램 시그널의 실시간성 부족 및 데이터 누락 (봇 정책 문제), 대시보드 내 불필요한 섹션 존재.

## 2. 향후 릴리즈 계획 (2026.03.05)

### Phase 1: 대시보드 미니멀라이즈 (Minimalism)

- [ ] 홈화면에서 "글로벌 뉴스", "AI Trade Journal", "하단 푸터 링크" 등 트레이딩에 직접 영향 없는 것들 삭제.
- [ ] 텔레그램 시그널, 경제 캘린더, 메인 차트(매물대 박스 포함) 3개 섹션에만 집중.

### Phase 2: 텔레그램 실시간성 확보 (Real-time Sync)

- [ ] 웹훅(Webhook) 방식 검토 혹은 MT5 시그널 파일 직접 동기화(Cloud Sync 활용) 구현.
- [ ] "실시간성"이 보장되지 않는 폴링 기반 코드 완전 제거.

### Phase 3: 차트 매물대 박스 정밀화

- [ ] MT5에서 가져온 H1 매물대 데이터를 기반으로 TradingView 차트 위에 데이트레이딩용 박스 영역 자동 생성.
