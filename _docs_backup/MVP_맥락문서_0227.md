## 2. MVP 부트캠프 프로젝트 결정 사항

### ❌ 탈락한 아이디어들
- 부업/사이드잡 큐레이션 → 도메인 경험 없음, 수동 큐레이션 부담
- AI 툴 큐레이션 → 경쟁자 이미 존재 (aipan.kr, carat.im)
- 금융상품 큐레이션 → 토스/네이버 경쟁 강함, 금리 하락기라 약함
- 트레이딩 커뮤니티 → 본인도 "큰 도움 안 됨", 한국트레이더협회가 이미 강자
- 부트캠프 큐레이션 → 부트텐트에 이미 있음, 수동 큐레이션 눈 부담

### ✅ 확정 방향
**"어머니를 위한 AI 말벗 앱"**

어르신 대상 AI 케어 앱. 한글 워크북과 별개로 MVP 부트캠프 프로젝트로 진행.

## 3. AI 말벗 앱 — 핵심 설계

### 타겟 유저 페르소나
- 스마트폰 기초 사용자 (전화, 카메라, 네이버)
- 타이핑 불가 → **음성과 터치만으로 작동해야 함**
- 혼자 계신 어르신

### MVP 핵심 기능 3가지 (1~2개월)
```
1. AI 말벗 대화 (음성)
   - 큰 버튼 하나 누르면 바로 대화 시작
   - 로그인 없음, 타이핑 없음
   - 일상 대화 → 치매 예방 효과

2. 약봉투 사진 찍기
   - Gemini Vision으로 인식
   - "이 약은 점심 후에 드세요" 음성으로 알려줌

3. 복지서비스 찾기 (추후)
   - 공공데이터 API 활용
   - "내가 받을 수 있는 지원이 뭐예요?" 음성 질문
```

### 수익화 로드맵
```
초기: 무료 (어머니 + 주변 어르신 테스트)
중기: 복지관/요양원 B2B
장기: 지자체 연계
```

- ECN: 실제 시장 직접 연결, 체결 투명성 높음
- 수익 EA 지속 시 STN은 브로커가 체결 질 저하 가능
- ECN은 구조적으로 브로커 개입 어려움
- A북 전환 시 ECN이 압도적으로 유리

**STN vs ECN 핵심 차이:**
```
STN: 브로커 내부 처리, 브로커가 반대편
     체결 거부/지연/슬리피지 조작 가능
     스프레드 = 브로커 수익

ECN: 실제 시장(LP)에 직접 전달
     체결은 시장이 결정
     브로커는 커미션만 수취
     원래부터 A북 구조
```

**다음 할 일:**
```

□ 인피녹스 ECN 커미션 확인 (?/lot)

□ 커미션 낮은 브로커로 실계좌 ECN 개설
□ VPS 셋업 (브로커 서버 인접)
```
